# Chicago Fire Watch

Independent live construction stream of the Chicago Fire FC stadium at The 78.

**Domain:** [chicagofirewatch.com](https://chicagofirewatch.com)

## Theme

Dear Chicago / Chicago School industrial heritage:
- Deep Fire red (`#C8102E`)
- Brick, steel, charcoal palette
- Timeless, proud, professional

## Stack

- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS v4
- Lucide React icons
- Ready for Vercel deploy

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Key Features (MVP)

- Live video player (YouTube embed + placeholder)
- Construction timeline
- Progress gallery
- Community & shop placeholders
- Fully responsive + dark industrial aesthetic
- Independent fan disclaimer
